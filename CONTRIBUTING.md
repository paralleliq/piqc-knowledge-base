# Contributing

Thank you for your interest in improving the GenAI Model Deployment Checklist.

This project is intended to remain:
- High-level
- Framework-neutral
- Infrastructure-agnostic

## How to Contribute

1. Open an Issue describing your suggestion or correction
2. Provide context or real-world examples if possible
3. Keep checklist items concise and actionable
4. Avoid vendor-specific recommendations

## Contribution Guidelines

- Use GitHub checklist syntax (`- [ ]`)
- One idea per checklist item
- Prefer operational validation over theory
- Do not add proprietary logic or algorithms
